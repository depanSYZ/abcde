## 🌟 Overview
**D2:业 API** adalah ekosistem REST API yang dirancang untuk kecepatan dan ketepatan data. Menggunakan arsitektur **Express.js** yang dioptimalkan untuk **Vercel Edge Functions**, memberikan respon *sub-second* untuk kebutuhan bot, web, maupun automasi.

## 🛠️ Tech Stack
* **Runtime:** Node.js (Vercel Node Runtime)
* **Framework:** Express.js (Monolithic Router)
* **Scraping:** Axios & Cheerio (High-Speed Extraction)
* **AI Logic:** Custom Handshake & Stream Processing
